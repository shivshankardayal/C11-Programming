Chapter 17. Errors <errno.h>

The header <errno.h> defines several macros, all relating to the reporting of error conditions.

The macros are:

EDOM EILSEQ ERANGE

which expand to integer constant expressions with type int, distinct positive values, and which are suitable for use in #if preprocessing directives; and errno which expands to a modifiable lvalue[150]


[150] The macro errno need not be the identifier of an object. It might expand to a modifiable lvalue resulting from a function call (for example, *errno()).
